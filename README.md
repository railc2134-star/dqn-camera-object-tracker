# RL Camera Tracker

A reinforcement learning based camera object tracker.
## What it is
This project uses reinforcement learning to track objects in a camera feed.
## What it does
The project trains a reinforcement learning model to track objects in a camera feed and move a crosshair to the center of the object.
## Technologies used
- Python
- PyTorch
- OpenCV
## How to run it
To run the project, make sure you have Python, PyTorch, and OpenCV installed. You can then run the script and it will start tracking objects in your camera feed.
